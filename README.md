## 🚀 快速开始

### 1. 上传文件
将本目录下的所有文件上传到虚拟主机的网站根目录（通常是 `public_html`、`www` 或 `htdocs`）

### 2. 设置权限
- 目录权限：755
- 文件权限：644

### 3. 访问网站
在浏览器中访问您的域名或IP地址

### 4. 登录后台
访问：`http://yourdomain.com/#/admin`
- 用户名：admin
- 密码：admin

**重要**：首次登录后请立即修改密码！

## ⚠️ 重要提示

### 安全建议

1. **修改默认密码**：首次部署后立即修改管理员密码
2. **定期备份**：定期下载 `site_data.json` 文件
3. **启用HTTPS**：建议配置SSL证书
4. **监控日志**：定期查看 `api_log.txt`

### 文件权限

- 根目录需要可写权限（用于生成数据文件）
- 如果无法保存数据，请设置目录权限为 755
